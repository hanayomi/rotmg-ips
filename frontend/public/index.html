How to Use the IP Tool

This tool allows you to quickly copy Realm of the Mad God server IPs to your clipboard. Simply click on the realm buttons under each server group. The IP will be copied, and a notification will let you know.

Frequently Asked Questions

What is a Nexus?

A Nexus is the central hub for the Realm of the Mad God servers where players gather before entering realms.

What are [BETA] realms?

[BETA] realms are testing environments for new features and content, allowing players to experience upcoming updates.

How do I search for a server?

Use the search box above to filter servers by name. Simply type in a server name or part of it, and the matching servers will be displayed.
